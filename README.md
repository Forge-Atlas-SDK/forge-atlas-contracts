# Forge Atlas Contracts

Core smart contracts powering the Forge Atlas programmable economy infrastructure.

## Status

The contracts are currently undergoing a security audit.

Audit completion is expected in **April 2026**, after which the audited contracts and documentation will be published in this repository.

## About Forge Atlas

Forge Atlas provides infrastructure that allows developers to:

• Create programmable game economies  
• Manage balances and spending rules  
• Enable player-owned assets  
• Settle economic activity onchain
